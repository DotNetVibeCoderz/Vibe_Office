# PersonalFinanceApp

Aplikasi Keuangan Pribadi (Personal Finance) dengan teknologi .NET MAUI Blazor Hybrid (WPF + Blazor).
Didesain dengan gaya **Brutalist** yang modern dan tegas.

## Fitur
- **Dashboard**: Ringkasan pemasukan, pengeluaran, saldo, dan transaksi terakhir.
- **Transactions**: Pencatatan transaksi lengkap dengan kategori dan akun.
- **Budgets**: Pengaturan anggaran bulanan dan monitoring.
- **Goals**: Target menabung dan tracking progress.
- **Reports**: Grafik analisis keuangan.
- **Security**: Login dengan PIN (Default: 1234).
- **Export**: Unduh data transaksi ke CSV.

## Teknologi
- **Frontend**: Blazor Hybrid (WPF host)
- **UI Framework**: MudBlazor
- **Database**: SQLite dengan Entity Framework Core
- **Charts**: LiveCharts2

## Cara Menjalankan
1. Pastikan .NET 8 SDK terinstall.
2. Buka project di Visual Studio atau VS Code.
3. Build dan Run project `PersonalFinanceApp`.
4. Database `finance.db` akan dibuat otomatis saat pertama kali dijalankan.
5. Login menggunakan PIN default: **1234**.

# English Description

Personal Finance Application built with .NET Blazor Hybrid (WPF + Blazor).
Designed with a modern **Brutalist** style.

## Features
- **Dashboard**: Summary of income, expense, balance, and recent transactions.
- **Transactions**: Record transactions with categories and accounts.
- **Budgets**: Set monthly budgets and monitor spending.
- **Goals**: Set savings goals and track progress.
- **Reports**: Financial analysis charts.
- **Security**: PIN Login (Default: 1234).
- **Export**: Download transaction data to CSV.

## Tech Stack
- **Frontend**: Blazor Hybrid (WPF host)
- **UI Framework**: MudBlazor
- **Database**: SQLite with Entity Framework Core
- **Charts**: LiveCharts2

## How to Run
1. Ensure .NET 8 SDK is installed.
2. Open the project in Visual Studio or VS Code.
3. Build and Run `PersonalFinanceApp`.
4. The database `finance.db` will be created automatically on first run.
5. Login using default PIN: **1234**.
